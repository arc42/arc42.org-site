The original organizing idea

Twelve drawers. Put each fact where people expect it.

The structure is deliberately boring in the best possible way: stable, learnable, and flexible enough for almost any software system.

A note from Peter“A good template should disappear behind the content. If people discuss the system instead of the document, it is doing its job.”
