Formats, defined

AsciiDoc
Powerful yet simple markup, used by arc42 itself. Ideally suited to architecture documentation, works well with docToolchain.
Markdown
Widespread, simple markup. Strict variants restrict it further; multi-page (MP) variants split every chapter into its own file.
Word (.docx)
Microsoft Word, also usable with LibreOffice or OpenOffice.
GitHub Markdown
GitHub Flavored Markdown, used on GitHub.com and GitHub Enterprise.
LaTeX
The document preparation system, for those who need beauty and are willing to invest some effort.
reStructuredText
Heavily used by Read the Docs and in the Python world.
Textile
Another simple markup language.
HTML
For viewing the template only, not for working with it.
Confluence
The commercial wiki by Atlassian; generated with help, flat or one page per section. For an up-to-date workflow we recommend authoring in AsciiDoc and syncing with asciidoc2confluence.
Tool formats
Enterprise Architect, IBM Rhapsody and Doxygen templates are available too, contributed by the arc42 community. All formats are generated from their AsciiDoc sources in the GitHub repository.

New to arc42?

Start with the 12 sections, pick the format with help, and delete what you do not need. Tailoring is expected, not cheating.
