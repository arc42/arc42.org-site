05 / Layout behavior

Wide enough for comparison, narrow enough for reading.

The system uses a maximum shell for archives and structured comparisons. Long prose remains capped at approximately 72 characters. Optional rails provide navigation or field notes without reducing reading comfort.

At smaller viewports, the visual hierarchy remains decisive. Controls grow to reliable touch sizes, side rails become inline content, and the interface never requires horizontal panning.
